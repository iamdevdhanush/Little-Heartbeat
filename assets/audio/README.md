# Audio Assets

## Heartbeat Sound

Place a `heartbeat.mp3` file here for the heartbeat audio feature.

### Recommended Audio:
- Duration: 10-15 seconds loop
- Format: MP3, 128-256kbps
- Volume: Normalized

### Where to get free heartbeat sounds:
1. Freesound.org - search "heartbeat"
2. SoundBible.com
3. YouTube Audio Library

### How to add:
1. Download a heartbeat/fetal heartbeat sound
2. Rename it to `heartbeat.mp3`
3. Place it in this folder
4. The app will automatically use it

Note: The app works without audio - it uses haptic feedback as a fallback.
