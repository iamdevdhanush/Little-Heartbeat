# Little Heartbeat Design System

## Typography
*   Primary Font: Inter
*   Headings: Inter SemiBold
*   Body: Inter Regular

## Colors
*   **Primary:** #4A3B4D (Slate Plum) - Used for primary text and high-emphasis elements.
*   **Secondary:** #5C7CFA (Dusk Blue) - Used for primary actions, active states.
*   **Background:** #FBFBFC (Pearl White) - Subtle cool off-white.
*   **Surface:** #FFFFFF (Pure White) - Used for cards.
*   **Success:** #34C759 (Sage Green)
*   **Warning:** #FF9F0A (Amber)
*   **Danger:** #FF453A (Coral Red)

## Layout & Spacing
*   Grid: 8px baseline
*   Card Padding: 24px
*   Border Radius: Cards 20px, Buttons 12px

## Components
*   **Primary Button:** Background #5C7CFA, Text #FFFFFF, Radius 12px, Padding 16px 24px.
*   **Card:** Background #FFFFFF, Border 1px solid rgba(0,0,0,0.06), Box-Shadow 0 2px 8px rgba(0,0,0,0.04).
